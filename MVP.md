# MVP Scope

## Included
- **Admin Dashboard**: List of forms, create new form.
- **Form Builder**: Title, description, thank-you message, question CRUD (Short Text, Long Text, MC, Checkbox, Dropdown, Rating).
- **Public Chat Runner**: Responsive chat interface for answering questions.
- **Responses View**: Table of all responses per form.
- **Export**: CSV export of responses.
- **Publishing**: Toggle between Draft and Published.

## Excluded (Phase 2+)
- Fake respondents generation.
- Advanced AI follow-ups (post-MVP).
- Custom themes per form (MVP will use a standard design from Stitch).
- File upload question type.
- Multi-step branching logic.
