# LimaChat

LimaChat is an AI-powered conversational form builder.

Instead of boring forms, users answer questions through a chat interface.

## 🚀 Features
- Chat-based survey experience
- Shareable public link
- Admin dashboard
- Response storage
- Clean UI

## 🌐 Live Demo
https://lima-chat.vercel.app/

## 🛠 Tech Stack
- Next.js
- Prisma
- PostgreSQL
- Vercel

## 📌 Vision
To make forms feel like conversations.
